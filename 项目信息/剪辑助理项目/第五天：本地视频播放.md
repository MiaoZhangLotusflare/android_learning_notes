# 剪辑助手项目 第四天：本地视频播放

　　今天完成的功能主要是本地视频的播放。

## 1. 概述
　　显示号本地视频的列表之后，就要实现视频的播放功能。

　　视频的播放选择了 VideoView 组件。

## 2. 使用 VideoVide 实现视频播放
```
    <VideoView
        android:id="@+id/activity_local_video_show_video_vv"
        android:layout_width="wrap_content"
        android:layout_height="0dp"
        android:layout_gravity="center_horizontal"
        android:layout_weight="1" />
````

　　播放：
```
   private void showVideo(String url) {
        vv_showVideo.setVideoURI(Uri.parse(url));
        vv_showVideo.start();
        tv_totalDuration.setText(timeIntToStr(localVideoBeanList.get(currentIndex).getDuration()));
        vv_showVideo.setOnCompletionListener(new MediaPlayer.OnCompletionListener() {
            @Override
            public void onCompletion(MediaPlayer mediaPlayer) {
                Log.d(TAG, "onCompletion");
                handler.removeMessages(MESSAGE_UPDATE);
                sb_seek.setProgress(100);
                tv_duration.setText(tv_totalDuration.getText());
            }
        });

        vv_showVideo.setOnErrorListener(new MediaPlayer.OnErrorListener() {
            @Override
            public boolean onError(MediaPlayer mediaPlayer, int i, int i1) {
                Log.d(TAG, "onError i:" + i + ",i1:" + i1);
                return false;
            }
        });

        handler.sendEmptyMessage(MESSAGE_UPDATE);
    }
```

## 总结
　　视频播放就使用了 VideoVide 组件，很快就实现了。
